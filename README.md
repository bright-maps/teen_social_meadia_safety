## About
This project is a social media awareness website designed to educate and protect teenagers while using social media platforms. It also serves as a guide for parents, guardians, and caregivers by providing practical knowledge on how to help children stay safe online. The website focuses on creating a safer digital environment by supporting both teenagers and the adults responsible for their well-being.

## Features
- Teen online safety tips
- Cyberbullying awareness and prevention
- Privacy protection guidance
- Safe social media practices
- Parental and guardian guidance
- Digital responsibility education
- Educational campaign content

## Purpose
To protect teenagers online while also empowering parents and guardians with the tools, knowledge, and strategies needed to monitor, guide, and support safe social media use. This project promotes shared responsibility between young users and caregivers for a safer online experience.

## Target Audience
- Teenagers and young social media users
- Parents and guardians
- Teachers and caregivers
- Community awareness groups

## Technologies Used
- HTML
- CSS
- JavaScript

## Vision
To build a safer online space where teenagers can enjoy social media responsibly while parents and guardians actively help protect and guide them.
